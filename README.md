1.Project title and overview
-**Title:** PAHub: AI-Powered CRM & Marketing Hub
-**Discription:** A modular Python application designed to provide free, accessible marketing tools for small businesses. it combines a Customer Relationship Management (CRM) system with a dynamic AI Marketing Engine.

2.System Architecture
-**logic layer:** Built with Python using a modular approach to seperate data handling fromthe user interface.
-**Data Layer:** Uses a lightweight JSON persistence layer for CPU-effiency, allowing the app to run on standard hardware without a heavy database.
-**AI Engine:** Adynamic prompting system that handles user-defined marketing topics.

3.Key Features 
-**Dynamic CRM:** Add and track contacts, companies, and deals with automatic timestamping.
-**Universal AI Content:** Generates personalized marketing emails for any user-inputted topic.
-**Responsible AI Governiance:** Hard-coded ethical disclaimers on all AI-generated content to ensure transparency.
-**Professional Exports:** Capability to export CRM data to CSV for external reporting.
